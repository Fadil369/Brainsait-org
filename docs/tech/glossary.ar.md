# قاموس المصطلحات التقنية

## A

### API (واجهة برمجة التطبيقات)
واجهة لتواصل مكونات البرمجيات.

### ASR (التعرف التلقائي على الكلام)
تحويل الكلام إلى نص.

## C

### CI/CD (التكامل المستمر/النشر المستمر)
خطوط أنابيب البناء والاختبار والنشر الآلية.

### Container (الحاوية)
بيئة تشغيل معزولة وخفيفة الوزن.

## D

### DICOM (التصوير الرقمي والاتصالات في الطب)
معيار التصوير الطبي.

### Docker
منصة حاويات لتغليف التطبيقات.

## F

### FHIR (موارد قابلية التشغيل البيني السريع للرعاية الصحية)
معيار تبادل بيانات الرعاية الصحية.

## H

### HL7 (المستوى الصحي السابع)
منظمة معايير قابلية التشغيل البيني للرعاية الصحية.

## K

### K3s
توزيعة Kubernetes خفيفة الوزن.

### Kubernetes
منصة تنسيق الحاويات.

## M

### mTLS (TLS المتبادل)
المصادقة بالشهادات ثنائية الاتجاه.

## N

### NLP (معالجة اللغة الطبيعية)
الذكاء الاصطناعي لفهم اللغة البشرية.

## O

### OAuth
إطار عمل التفويض للوصول إلى API.

### OCR (التعرف الضوئي على الحروف)
تحويل الصور إلى نص.

## R

### RAG (التوليد المعزز بالاسترجاع)
تقنية ذكاء اصطناعي تجمع بين البحث والتوليد.

### REST (نقل الحالة التمثيلية)
نمط بنية API.

## S

### SLA (اتفاقية مستوى الخدمة)
التزامات الأداء.

## T

### TLS (أمان طبقة النقل)
تشفير اتصالات الشبكة.

### TTS (تحويل النص إلى كلام)
تحويل النص إلى صوت منطوق.

## V

### VPN (الشبكة الخاصة الافتراضية)
نفق شبكة مشفر.
