# نظرة عامة على API

## نظرة عامة

توفر BrainSAIT واجهات برمجة تطبيقات RESTful للتكامل مع منصتنا. تستخدم جميع واجهات API صيغة JSON وتتبع مواصفات OpenAPI 3.0.

---

## عناوين URL الأساسية

| البيئة | العنوان |
|--------|---------|
| الإنتاج | `https://api.brainsait.com/v1` |
| التجربة | `https://sandbox.brainsait.com/v1` |

---

## المصادقة

تتطلب جميع واجهات API المصادقة عبر:
- مفتاح API (في الترويسة)
- رمز OAuth 2.0 Bearer

انظر [المصادقة](authentication.ar.md) للتفاصيل.

---

## واجهات API الأساسية

### واجهة المطالبات API
- تقديم المطالبات
- التحقق من الحالة
- الحصول على النتائج

### واجهة الأهلية API
- التحقق من التغطية
- التحقق من المنافع

### واجهة المستندات API
- رفع المستندات
- معالجة الاستخراج
- الحصول على النتائج

### واجهة التحليلات API
- استعلام المقاييس
- إنشاء التقارير
