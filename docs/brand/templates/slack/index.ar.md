## البداية السريعة

1. **اختر قالبًا** من الفئات أعلاه
2. **راجع الهيكل** وخصصه حسب احتياجاتك
3. **أنشئ قناة Slack** باستخدام إرشادات القالب
4. **استورد إلى Notion** باستخدام تنسيق Markdown

## الاستخدام مع Notion

جميع القوالب منسقة للاستيراد السهل إلى Notion:

1. انسخ محتوى Markdown
2. في Notion، استخدم **استيراد** > **Markdown**
3. عدّل التنسيق حسب الحاجة
4. أعد خصائص قاعدة البيانات من البيانات الوصفية

## التخصيص

يتضمن كل قالب:

- **البيانات الوصفية**: بيانات للتصنيف
- **هيكل القنوات**: القنوات والأغراض الموصى بها
- **خطوات سير العمل**: التنفيذ على مراحل
- **الرسائل الرئيسية**: رسائل الترحيب وقوالب المتابعة
- **أفضل الممارسات**: توصيات Brainsait
- **نقاط التكامل**: الاتصال مع وكلاء Brainsait
