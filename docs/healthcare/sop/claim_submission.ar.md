# إجراء تقديم المطالبة

## الغرض

هذا الإجراء يحدد الخطوات القياسية لتقديم المطالبات عبر نفيس.

## النطاق

ينطبق على جميع موظفي الفوترة ومقدمي الرعاية الصحية.

## الإجراء

### 1. التحقق من الأهلية

- تسجيل الدخول إلى نظام نفيس
- إدخال رقم هوية المريض
- التحقق من التغطية النشطة
- تأكيد الخدمات المشمولة

### 2. توثيق الزيارة

- تسجيل جميع التفاصيل السريرية
- إرفاق الوثائق الداعمة
- الحصول على موافقة الطبيب

### 3. الترميز

- تعيين رموز ICD-10-AM للتشخيص
- تعيين رموز ACHI للإجراءات
- التحقق من دقة الرموز

### 4. التحقق من المطالبة

- استخدام ClaimLinc للتحقق
- مراجعة جميع التحذيرات
- تصحيح أي أخطاء

### 5. التقديم

- إنشاء حزمة FHIR
- تقديم عبر نفيس API
- حفظ رقم التقديم

### 6. المتابعة

- مراقبة حالة المطالبة
- الرد على طلبات المعلومات
- معالجة الرفض إذا لزم الأمر

## المسؤوليات

- **موظف الفوترة**: تقديم المطالبة
- **الطبيب**: الموافقة السريرية
- **المدير**: مراجعة الجودة

## الوثائق المطلوبة

- تقرير الزيارة
- نتائج المختبر (إن وجدت)
- صور الأشعة (إن وجدت)
- التفويض المسبق (للخدمات المحددة)
